Abstract

The Data Cube vocabulary allows to publish statistical data on the web, using the W3C RDF (Resource Description Framework) standard. The QB+ vocabulary is an extension to the Data Cube vocabulary that allows to represent OLAP cubes in RDF, and also allows to implement OLAP operators (such as Roll-up, Slice, and Dice) as SPARQL queries on this RDF representation.
